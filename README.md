# Project_Work_at_Campus

This repository contains the source code for a Unity game demo.

Supported Unity version: `Unity 2022.3.62f2c1` (2022.3 LTS).

You can open and build this project using the Unity Editor (via Unity Hub or by launching the Editor directly).
